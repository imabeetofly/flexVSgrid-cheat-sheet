flexbox cheat-sheet

display: flex > set the display to flex (every direct child will become a column even if there is overflow) link

flex-wrap: ... > sets whether flex items are forced onto one line or can wrap onto multiple lines. link

align-items: ... > it controls the alignment of items on the Cross Axis. link

justify-content: ... > defines how the browser distributes space between and around content items along the main-axis of a flex container. link

Grid cheat-sheet

display: grid > set the display to grid (every direct child will stay a row) link

grid-auto-flow: ... > change the behavior of the grid (row/column) link

grid-template-row: ... > defines the line names and track sizing functions of the grid columns. link

grid-template-row: 1fr 1fr 1fr 1fr (create 4 colums of the same lenght)

grid-template-row: repeat(4, 1fr) (create 4 colums of the same lenght)

grid-template-columns: reapeat(auto-fit, minmax(300px,1fr)) (create colums of min 300px of lenght and keep all the columns they can feet at the same lenght )

grid-template-rows: ... > defines the line names and track sizing functions of the grid rows. link

ups- and downs of Grid

  • for structured layout grid is a much better solution
